# Customer Data Analysis Project

## Project Description

This project demonstrates a complete customer data analysis workflow using Python, Pandas, NumPy, SQLAlchemy, PostgreSQL, and Power BI. The objective is to analyze customer purchasing behavior, clean and preprocess raw data, identify trends, and present actionable insights through interactive dashboards.

The project follows a real-world data analytics pipeline, starting from data collection and cleaning, followed by exploratory data analysis (EDA), database integration, and business intelligence visualization.

## Key Features

* Imported and explored customer transaction data.
* Performed data cleaning by handling missing values, removing duplicate records, and correcting inconsistent data.
* Validated data quality using logical comparisons and consistency checks.
* Removed redundant columns after verifying duplicate information.
* Conducted exploratory data analysis (EDA) to identify customer purchasing patterns.
* Calculated descriptive statistics and generated business insights.
* Connected Python with PostgreSQL using SQLAlchemy.
* Stored cleaned datasets in a PostgreSQL database for efficient data management.
* Built interactive Power BI dashboards to visualize customer behavior and business performance.
* Documented the complete analysis process for reproducibility and future improvements.
